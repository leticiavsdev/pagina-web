01 - História da Internet

  • Surgimento da internet (ARPANET)
  • Evolução até a web (WWW)
  • Importância da internet hoje
  • Conceito de cliente x servidor

Neste capítulo é apresentado como a internet surgiu, começando com a ARPANET, um projeto militar dos Estados Unidos criado para permitir a comunicação entre computadores mesmo em situações de guerra. Com o tempo, essa rede foi evoluindo e se expandindo para universidades e depois para uso comercial. Também é explicado como a criação da World Wide Web (WWW) facilitou o acesso às informações por meio de páginas.
👉 Exemplo: antes da web, não existiam sites como conhecemos hoje — só troca de dados entre máquinas.
👉 A ideia principal é entender que a internet não nasceu pronta, ela foi construída ao longo do tempo.

03 - Como funciona HTML e CSS

  • HTML = estrutura (conteúdo)
  • CSS = aparência (estilo)
  • Separação de responsabilidades
  • Funcionamento básico das páginas

Esse capítulo explica a base do desenvolvimento web: HTML e CSS. O HTML é responsável pela estrutura do conteúdo (títulos, textos, imagens), enquanto o CSS cuida da aparência (cores, fontes, layout). Também reforça a importância de separar conteúdo de estilo.
👉 Exemplo: o HTML cria um texto, o CSS deixa ele bonito.
👉 Aqui você começa a entender como um site é construído de verdade.

04 - Primeiros passos HTML

  • Estrutura básica (<!DOCTYPE>, <html>, <head>, <body>) 
    
              <!DOCTYPE html>
              <html lang="pt-br">
              <head>
                <meta charset="UTF-8">
                <title>Título</title>
              </head>
              <body>
              </body>
              </html>

head → configurações

body → conteúdo visível

  • Primeiras tags
  • Criando seu primeiro site

Neste ponto, você aprende a criar a estrutura básica de um documento HTML, com tags como <html>, <head> e <body>. Também entende o papel do <!DOCTYPE html> e da codificação de caracteres (UTF-8).
👉 Exemplo: qualquer site começa com essa estrutura básica antes de adicionar conteúdo.
👉 Esse capítulo é essencial porque é o primeiro contato prático com código.

05 - Caracteres, parágrafos e quebras

  • Uso de <p>,<br>
  • Caracteres especiais (HTML entities)
  • Espaçamento no HTML

Aqui você aprende a trabalhar com textos no HTML, usando <p> para parágrafos e <br> para quebra de linha. Também são apresentados caracteres especiais (como os simbolos < e >) e emojis.
👉 Exemplo: escrever um texto organizado em parágrafos melhora a leitura.
👉 O foco é ensinar como estruturar conteúdo textual corretamente.

06 - Imagens e Favicon

  • Inserir imagens (<img>)
  • Caminhos (relativo/absoluto)
  • Favicon (ícone do site)

Esse capítulo ensina como inserir imagens usando a tag e a importância do atributo alt. Também mostra como definir o favicon, que é o ícone da aba do navegador.
👉 Exemplo: 

<img src="foto.jpg" alt="Foto de perfil">;
👉 Você aprende a trabalhar com caminhos de arquivos e organização de pastas.

07 - Hierarquia de títulos

  • Uso correto de <h1> até <h6>
  • Importância semântica
  • Organização do conteúdo

Aqui é explicado como usar corretamente os títulos (<h1> até <h6>) para organizar o conteúdo. A hierarquia ajuda tanto na leitura quanto no SEO.
👉 Exemplo: <h1> para título principal, <h2> para subtítulos.
👉 Esse capítulo reforça a importância da estrutura lógica do conteúdo.

08 - Formatação de textos

  • Negrito (<strong>)
  • Itálico (<em>)
  • Marcação semântica
  • Diferença entre visual e significado

Neste capítulo você aprende a diferenciar formatação visual de semântica. Tags como e têm significado, diferente de <b> e <i>.
👉 Exemplo: <strong> indica importância, não só negrito.
👉 O foco é criar um código mais correto e acessível.

09 - Listas HTML

  • Listas ordenadas (<ol>)
  • Listas não ordenadas (<ul>)
  • Listas de definição (<dl>)

Aqui você aprende a criar listas ordenadas (<ol>), não ordenadas (<ul>) e itens (<li>). Também são introduzidas listas de definição.
👉 Exemplo: listas são muito usadas em menus e conteúdos organizados.
👉 Esse capítulo ajuda a estruturar melhor informações no site.

10 - Ligações (links)

  • Links com <a>
  • Links internos e externos
  • Âncoras (navegação dentro da página)

Esse capítulo ensina a usar a tag para criar links entre páginas ou para sites externos. Também mostra como abrir links em nova aba.
👉 Exemplo: 

<a> href="pagina.html">Ir</a>
👉 Links são a base da navegação na web.

11 - Imagens, áudios e vídeos

  • Responsividade de imagens
  • Inserção de áudio e vídeo
  • Uso de <audio> e <video>

Aqui você aprende a adicionar áudio e vídeo com <audio> e <video>, além de trabalhar melhor com imagens responsivas.
👉 Exemplo: um site pode ter um vídeo incorporado diretamente.
👉 O objetivo é deixar o site mais rico e interativo.

📜 PARTE 1 — COLINHA DE TAGS HTML (com explicação)

📜 Estrutura básica

  • <html> → raiz do site
  • <head> → configurações
  • <body> → conteúdo

📌 Texto e conteúdo

  • <h1> até <h6>; → títulos
  • <p> → parágrafo
  • <br> → quebra de linha ←
  • <hr> → linha horizontal

🎓 Formatação (semântica)

  • <strong> → importante
  • <em> → ênfase
  • <mark> → destaque
  • <small>; → texto pequeno
  • <del> → texto riscado
  • <ins> → texto inserido sublinhado, para alguma citação
  • <sup> Deita a palava em cima
  • <sub> Deita a palava em baixo
  • <a> adiciona link

🎓 Listas

  • <ul> → lista com pontos
  • <ol> → lista numerada
  • <li> → item da lista
  • Links <a> → link href → destino target="_blank" → nova aba

🎓 Mídia

  • <img> → imagem
  • <audio> → áudio
  • <video> → vídeo
  • <source> → fonte do arquivo

🎓 Organização (semântica moderna)

  • <header> → topo
  • <main> → conteúdo principal
  • <section> → seção
  • <article> → conteúdo independente
  • <footer> → rodapé

🎓 Outras úteis

  • <div> → bloco genérico
  • <span> → linha (inline)
  • <code> → código
  • <pre> → Explicar codigo
  • <style> → Colocar extilo CSS
  • <andress> → Colocar endereço
  • <abbr> → Informa o significado das abreviações
